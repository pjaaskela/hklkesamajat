GDPR & tietosuoja

Kerättävät henkilötiedot: nimi, mökin numero, sähköposti, puhelin, kotiosoite, rooli

Oikeusperuste: yhdistyslaki (jäsenrekisteri), sopimus (maanvuokra), oikeutettu etu (alueen hallinnointi)

Tietosuojaseloste: tietosuoja.html

Kävijäseuranta: anonymisoitu IP, sivulataukset, kesto — ei kolmannen osapuolen evästeitä

Säilytysajat: jäsentiedot jäsenyyden ajan + 2v, kävijädata 90pv, vikailmoitukset 3v

Ylläpito-ohjeet

Demo-deploy (.36): scp tiedosto.html 192.168.86.36:/home/pjaaskel/web/

Julkinen deploy (hklkesamajat.fi): git push → cPanel Git pull

API uudelleenkäynnistys: ssh .33 "sudo systemctl restart hkl-api"

Drive-synkronointi: Admin-paneeli → Dokumentit → "Synkronoi nyt" tai POST /api/hkl/drive-sync

Tietokannan varmuuskopio: ssh .33 "cp /home/pjaaskel/web/data/hkl.db ~/backup/"
